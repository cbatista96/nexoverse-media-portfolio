📸 IMAGEN DE PERFIL
====================

Coloca tu foto en esta carpeta con el nombre exacto:

    carlos.jpg

Recomendaciones:
- Formato: JPG o PNG (renombrar a .jpg si es PNG)
- Tamaño: mínimo 800x1000 px (proporción 4:5)
- Calidad: alta resolución
- Recorte: rostro centrado, mirando a la cámara
- Fondo: limpio o neutro funciona mejor

La foto que enviaste con el traje granate y fondo claro es PERFECTA.
Solo guárdala en esta carpeta como "carlos.jpg" y se verá automáticamente.

Si usas otro nombre, edita la línea correspondiente en styles.css:
    .portrait__img { background-image: url('assets/carlos.jpg'); ... }
